This repository contains the dataset used for the Educational Website Web Tracking Project. It provides structured information about trackers used across educational websites over multiple years, along with derived graph representations.

**Repository Structure**
1. tracker_info.csv

This file contains metadata about trackers, including:

Tracker name/identifier
Provider (organization responsible for the tracker)
Category (e.g., analytics, advertising, social media, etc.)

This file serves as a reference table to interpret tracker identities and their roles.

2. TrackerData_[year].csv files:

These files contain the main dataset with tracking information collected from educational websites across different years. This data was used for the longitudinal analysis of tracking behavior in educational domains.
